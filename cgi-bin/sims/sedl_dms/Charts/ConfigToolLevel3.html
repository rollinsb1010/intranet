FusionCharts Visual Configuration Tool> Level 3 Properties

We will look at the properties based on the tabs in which they are present. In Level 3, the tabs that are present are shown below.

Level 3 Tabs

Tab Name Description
Pie/ Doughnut Configure the pie or doughnut charts using the properties in this tab.
Margin/ Legend Define margins and customize the padding of various elements on the chart.
Configure the functional and cosmetic properties of the legend.
Define Styles Define styles that can be applied to various objects.
Apply Styles Apply the defined styles to objects.
Miscellaneous Configure other properties specific to a few charts.

Each tab may have basic and advanced properties. We will now try to understand all the fields placed in these tabs. Note that the options Yes/No taken from the Value List YesNoList are internally converted to 0/1 to suit the FusionCharts QueryString format.

Plot Space Percent plotSpacePercent

On a column chart, there is spacing defined between two columns. By default, the spacing is set to 20% of canvas width. If you intend to increase or decrease the spacing between columns, you can do so using this attribute. For example, if you wanted all columns to stick to each other without any space in between, you can set plotSpacePercent to 0. Similarly, if you want very thin columns, you can set plotSpacePercent to its max value of 80.

On a bar chart, there is spacing defined between two bars. By default, the spacing is set to 20% of canvas width. If you intend to increase or decrease the spacing between bars, you can do so using this attribute. For example, if you wanted all bars to stick to each other without any space in between, you can set plotSpacePercent to 0. Similarly, if you want very thin bars, you can set plotSpacePercent to its max value of 80.

0-80 (in percent) Bar2D, Column2D, Column3D, MS Bar2D, MS Bar3D, MS Column2D, MS Column3D, Stacked Bar 2D, Stacked Column2D, Stacked Column3D, StackedColumn3DLineDualY, Combi2D, Combi2DDY, Column3DLine, Column3DLineDualY, ScrollCOlumn2D, ScrollCombi2D, ScrollCombiDY2D, ScrollStackedColumn2D

Glow, Bevel, Shadow Properties
BlurX blurX

The amount of horizontal blur. The default value is 4 for Bevel and Shadow, 8 for Glow. Values that are a power of 2 (such as 2, 4, 8, 16 and 32) are optimized to render more quickly than other values.

 0 to 255. Glow, Bevel, Shadow
BlurY blurY

The amount of vertical blur. The default value is 4 for Bevel and Shadow, 8 for Glow. Values that are a power of 2 (such as 2, 4, 8, 16 and 32) are optimized to render more quickly than other values.

 0 to 255 Glow, Bevel, Shadow
Strength strength

The strength of the imprint or spread. The higher the value, the more color is imprinted and the stronger the contrast between the glow/bevel/shadow and the background. The default is 1 for Bevel and Shadow, 2 for Glow.

 0 to 255 Glow, Bevel, Shadow
Quality quality

The number of times to apply the shadow effect. The default value is 1 for Bevel and Shadow, 2 for Glow, which is equivalent to low quality. A value of 2 is medium quality, and a value of 3 is high quality. Lower values are rendered quicker

 0-15 Glow, Bevel, Shadow
Angle angle The angle of the shadow. The default value is 45 0-360 Bevel, Shadow
Distance distance The offset distance for the shadow (in pixels). The default value is 4 Number (in pixels) Bevel, Shadow
Alpha alpha / shadowAlpha The alpha transparency value for the shadow color. For example, 25 sets a transparency value of 25%. 0-100 Glow, Bevel, Shadow
Color color / shadowColor The color of the glow/bevel/shadow. The default value is CCCCCC for shadow, FF0000 for Glow and for Bevel. Hex code without the # Glow, Bevel, Shadow
Highlight Color highlightColor

The highlight color of the bevel.The default value is FFFFFF

 Hex code without the # Bevel
Highlight Alpha highlightAlpha

The alpha transparency value of the highlight color. For example, 25 sets a transparency value of 25%. The default value is 50.

Apply Styles Tab

Apply Styles tab: Apply the styles to various chart objects. The Chart Objects are BACKGROUND, CANVAS, CAPTION, DATALABELS, DATAPLOT, DATAVALUES, DIVLINES, HGRID, LEGEND, SUBCAPTION, TOOLTIP, TRENDVALUES, VLINES, XAXISNAME, YAXISNAME, YAXISVALUES.

Select a object from the pop-up menu.
Click on a Style Name in the left side portal to select it, click on the + button to apply the style to the selected object.
Internally, the tool will create style,application, applyTo tags and set the toObject attribute value to the object selected from the pop-up menu and add the values of selected styles to the styles attribute.
