FusionCharts Visual Configuration Tool> Level 1 Properties

We will look at the properties based on the tabs in which they are present. In Level1, the tabs that are present are shown below.

Level 1 Tabs

Tab Name Description
General Chart level configuration can be done in this tab. The type, width and height of the chart, caption, sub-caption, x-axis name etc that need to be displayed on the chart, can be configured from this tab.
Labels In this tab, you can modify whether the labels, values have to be shown on the chart or not, also configure how they appear on the chart
Number We can configure the way, the numbers, look in the chart by changing the properties in this tab.
Look & Feel We can control the way the chart and canvas looks by changing the properties in this tab.
Script Variables Sometimes, we would like to pick the values for some attributes from script variables present in the caller application. This can be achieved by setting values in this tab.

Each tab may have basic and advanced properties. We will now try to understand all the fields placed in these tabs. Note that the options Yes/No taken from the Value List YesNoList are internally converted to 0/1 to suit the FusionCharts QueryString format.

Script Variables Tab : Assign Values From Your Application Using Script Variables
Script Variables any attribute

This is a special field. In this field, you can specify any attribute with the value coming from a script variable in the caller application. Basically, whatever is entered in this field as value, will remain un-processed by the tool. It is appended to the attribute list as it is, without any modification. On returning to the caller application, if the same script variable is found with a value, that will get automatically substituted and the chart will show up with those values.

Commonly, this field can be used to specify Caption, subCaption etc which are specific to the caller application and take values based on some condition in the caller application.

Example: SubCaption=$ProductName where caption is the attribute to which we want to assign the value $ProductName is a variable accessible in the script of the caller application.

Note that the preview of the chart in the tool will not show the values for these variables.

String in the format attrName=
$ScriptVariable

Depends on the attribute being specified. Please check the documentation to see if the attribute specified is supported by the selected chart type.
