FusionCharts Visual Configuration Tool> Dataset Properties

Let us take a look at the Dataset Properties. There are two tabs in the DataSet Properties Layout. Let us take a look at these:

DataSet Properties Tabs

Tab Name Description
General Configure the general properties of the dataset by setting values in this tab.
Anchor In this tab, you can modify the look and feel of anchors (if any)

We will now try to understand all the fields placed in these tabs. Note that the options Yes/No taken from the Value List YesNoList are internally converted to 0/1 to suit the FusionCharts QueryString format. "Chart Types Supporting This Property" indicates the Chart types ( among the list of charts presented by the tool ) which support the property.

The List of charts for which the dataset properties can be configured are: MS Column2D, MS Column3D, MS Line2D, MS Area2D, MS Bar2D, MS Bar3D, StackedColumn3D, StackedColumn2D, StackedArea2D, StackedBar2D, StackedBar3D, StackedColumn3DLineDualY, MS Combi2D, MS CombiDY2D, Column3DLineSingleY, Column3DLineDY, ScrollStackedColumn2D, ScrollCombi2D, ScrollCombiDY2D.

Note: The final block of parameters will not contain Chart Type and DataSet Number To Configure. These are used by the tool internally to generate a better preview of your dataset.
